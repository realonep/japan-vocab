마이크 권한 설정 가이드

권한 상태를 확인한 뒤 다시 테스트해 주세요.

Safari 설정 순서

  1. `Safari`에서 사이트 열기 → 주소창 좌측 `aA`(또는 사이트 설정)
  2. `웹사이트 설정` → `마이크` 진입
  3. `허용` 또는 `묻기`로 설정
  4. 페이지 새로고침 후 TEST 버튼 다시 실행

Chrome(iOS) 설정 순서

  1. iOS `설정` → `Chrome` → `마이크` 켜기
  2. Chrome에서 사이트 재접속 후 권한 팝업이 뜨면 `허용`
  3. 팝업이 안 뜨면 사이트 데이터 삭제 후 재시도
  4. 필요 시 Safari에서도 동일 사이트 권한 확인
